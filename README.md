# Linux StudEx

Linux StudEx project - Multi-VM Agent Platform

## Overview
A comprehensive multi-VM agent platform running on QEMU/UTM with:
- Remote Virtual Screens (RVS) for each agent VM
- Command Centre dashboard for centralized control
- Ollama local LLM inference
- LM Studio integration
- Multiple AI agents (Cashclaw, Hermes, OpenHuman, Cursor)

## VMs
| VM | Size | Agent |
|----|------|-------|
| RVS1 | 20GB | Cashclaw Agent |
| RVS2 | 20GB | Hermes Agent |
| RVS3 | 20GB | OpenHuman Platform |
| RVS4 | 20GB | Cursor IDE |
| RVS5-14 | 5GB each | Agent Farm |
