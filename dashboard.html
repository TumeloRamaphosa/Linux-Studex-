Quick Start

Get your Agent OS running in minutes.

# 1. Install tmux (already installed: v3.6a)
# 2. Configure tmux for agent monitoring:
cp .tmux.conf ~/.tmux.conf
tmux source ~/.tmux.conf

# 3. Start the master session:
tmux new -s linux-studex

# 4. Create windows for each agent:
tmux new-window -t linux-studex:1 -n cashclaw
tmux new-window -t linux-studex:2 -n hermes
tmux new-window -t linux-studex:3 -n openhuman
tmux new-window -t linux-studex:4 -n cursor
tmux new-window -t linux-studex:5 -n farm

# 5. Deploy agent VMs:
./orbit --deploy all

# 6. Open the dashboard:
open dashboard.html
